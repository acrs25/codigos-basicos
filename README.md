# 🐍 Códigos Básicos em Python

Este repositório contém **scripts simples em Python** criados para praticar lógica de programação e conceitos fundamentais da linguagem.  
Cada arquivo resolve um exercício específico, com foco em clareza e aprendizado.

## 📂 Exercícios disponíveis

- `contador-palavras.py` → Conta quantas palavras existem em uma frase.
- `fizz-buzz.py` → Implementa o clássico desafio FizzBuzz.
- `frequencia-letras.py` → Calcula a frequência de cada letra em uma palavra ou frase.
- `inverter-palavra.py` → Inverte uma palavra ou frase.
- `media-notas.py` → Calcula a média de notas de um aluno.
- `numero-primo.py` → Verifica se um número é primo.
- `palindromo-lista.py` → Verifica palíndromos em uma lista de palavras.
- `palindromo-palavra.py` → Verifica se uma palavra é palíndromo.
- `pares-impares.py` → Números pares e ímpares.
- `remover-duplicados.py` → Remove elementos duplicados de uma lista.
- `saudacao-personalizada.py` → Exibe uma saudação personalizada.
- `tabuada.py` → Gera a tabuada de um número.
